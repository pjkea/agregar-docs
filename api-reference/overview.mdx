## Authentication in the explorer

Click **Authorize** in the API explorer and enter either:
- Your API key in the `X-API-Key` field, or
- A Bearer token (get one from `POST /auth/login`)

All try-it calls go directly to the production API. Use the sandbox environment for testing.
